# Phonebook Management System

A simple command-line Phonebook Management System built using C++.

The application allows users to:

- Add contacts
- View contacts
- Search contacts
- Delete contacts

This project is useful for learning C++ fundamentals such as classes, vectors, functions, and basic CRUD operations.

## Features

- Add a new contact
- View all saved contacts
- Search for a contact by name
- Delete a contact
- Menu-driven interface
- Easy to use command-line application

  ## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- STL Vector
- Command Line Interface (CLI)
